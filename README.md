# ML with Python – Data Preprocessing Assignment

This repository contains solutions to fundamental data preprocessing tasks in Machine Learning using Python, Pandas, NumPy, and Scikit-learn.

## Objectives

The purpose of this assignment is to understand and implement common data preprocessing techniques that are essential before training Machine Learning models.

## Implemented Programs

| Question | Topic                                             |
| -------- | ------------------------------------------------- |
| Q1       | Missing Value Detection and Handling              |
| Q2       | Duplicate Data Detection and Removal              |
| Q3       | Min-Max Normalization                             |
| Q4       | Z-Score Standardization                           |
| Q5       | Outlier Detection using Z-Score                   |
| Q6       | Label Encoding                                    |
| Q7       | One-Hot Encoding                                  |
| Q8       | Feature Scaling using StandardScaler              |
| Q9       | Comparison of Min-Max Scaling and Standardization |
| Q10      | Numeric String to Integer Conversion              |

## Technologies Used

* Python 3
* Pandas
* NumPy
* Scikit-learn


## Learning Outcomes

After completing this assignment, I learned how to:

* Handle missing values in datasets
* Detect and remove duplicate records
* Normalize and standardize numerical data
* Identify outliers using statistical methods
* Encode categorical variables for Machine Learning
* Scale features using Scikit-learn preprocessing tools
* Convert and manage data types effectively
